Политика конфиденциальности

Дата вступления в силу: 1 марта 2026 г.

1. Какие данные мы собираем

При использовании Telegram-бота @TenderAI111_bot мы собираем:

  • Telegram user ID и имя пользователя (Telegram передаёт их автоматически при старте бота)
  • Настройки поисковых фильтров, которые вы вводите самостоятельно (ключевые слова, регионы, ценовые диапазоны)
  • Историю уведомлений о тендерах (номера тендеров, дата отправки)
  • Данные профиля компании, которые вы вводите самостоятельно (при заполнении)

2. Для чего используются данные

  • Поиск и фильтрация тендеров на zakupki.gov.ru по вашим настройкам
  • Отправка уведомлений о найденных тендерах
  • Улучшение точности AI-анализа релевантности
  • Статистика использования сервиса (анонимизированная)

3. Хранение данных

Данные хранятся на защищённых серверах в облачной инфраструктуре (Railway / AWS). Данные не передаются третьим лицам, кроме случаев, прямо предусмотренных настоящей политикой.

4. AI-анализ

Для проверки релевантности тендеров используется API OpenAI (GPT-4o-mini). В запросы к OpenAI передаётся только текст объявления о тендере с официального сайта zakupki.gov.ru. Персональные данные пользователей в запросы к OpenAI не включаются.

5. Google Sheets (при подключении)

Если вы подключаете интеграцию с Google Sheets, бот получает доступ только к указанной вами таблице. Доступ используется исключительно для записи данных о тендерах. Токен хранится в зашифрованном виде.

6. Ваши права

  • Вы можете удалить все свои данные, отправив команду /delete_account в боте или написав нам.
  • Вы можете запросить выгрузку своих данных, написав на контактный email.

7. Cookies и аналитика

Лендинг-страница использует Яндекс.Метрику для анонимной статистики посещаемости. Данные аналитики не связываются с персональными данными пользователей Telegram-бота.

8. Контакты

По вопросам конфиденциальности пишите в Telegram: @TenderAI111_bot

9. Изменения политики

Мы можем обновлять настоящую политику. Актуальная версия всегда доступна по этому адресу. При существенных изменениях вы получите уведомление в боте.
