# Быстрый старт

## Шаг 1: Установка (5 минут)

```bash
# Установите базовые зависимости
pip install PyPDF2 python-docx pyyaml jinja2 tqdm python-dotenv colorama requests

# Установите LLM провайдер (выберите один):

# Вариант 1: Groq (БЕСПЛАТНО, рекомендуется)
pip install groq

# Вариант 2: OpenAI (платный, дешевый)
pip install openai

# Вариант 3: Google Gemini (бесплатный tier)
pip install google-generativeai

# Вариант 4: Ollama (локально, бесплатно)
# Скачать: https://ollama.ai/

# Вариант 5: Anthropic Claude (оригинал, платный)
pip install anthropic
```

### Настройка API ключа

**Для Groq (рекомендуется):**
```bash
# 1. Получите ключ: https://console.groq.com/ (бесплатно)
# 2. Создайте .env файл:
echo "GROQ_API_KEY=gsk_ваш_ключ" > .env
```

**Для других провайдеров:** см. [LLM_PROVIDERS.md](LLM_PROVIDERS.md)

## Шаг 2: Настройка профиля компании (10 минут)

Отредактируйте `config/company_profile.yaml`:

```yaml
company_info:
  name: "Ваша Компания"

capabilities:
  categories:
    - "IT-оборудование"
    - "Разработка ПО"
    # Добавьте ваши категории

financial:
  cost_structure:
    labor_cost_per_hour: 2000  # Ваша ставка
    profit_margin_target: 0.25  # Желаемая маржа (25%)
```

## Шаг 3: Запуск (2 минуты)

```bash
# Поместите тендерную документацию в data/tenders/
# Затем запустите:

python main.py data/tenders/документация.pdf

# Или проанализируйте все файлы:
python main.py --path data/tenders/

# HTML отчет автоматически откроется в браузере
```

## Что вы получите

1. **HTML отчет** — красивая визуализация с оценками и рекомендациями
2. **JSON файл** — данные для программной обработки
3. **Markdown файл** — текстовая версия для чтения

## Результаты анализа

- ✅ **Итоговая оценка** (0-100 баллов)
- ✅ **Готовность к участию** (в процентах)
- ✅ **Рекомендация** (УЧАСТВОВАТЬ / УТОЧНИТЬ / ОТКАЗАТЬСЯ)
- ✅ **Список пробелов** с критичностью
- ✅ **Готовые вопросы** для заказчика
- ✅ **Контакты** заказчика
- ✅ **Финансовый анализ** (маржа, ROI, обеспечение)

## Примеры команд

```bash
# Один файл
python main.py tender.pdf

# Несколько файлов
python main.py doc1.pdf doc2.docx spec.pdf

# Все файлы в директории
python main.py --path /path/to/tender/docs/

# Без автоматического открытия браузера
python main.py tender.pdf --no-browser
```

## Где найти отчеты

Все отчеты сохраняются в `output/reports/` с именами вида:
```
tender_report_<название_тендера>_<timestamp>.html
tender_report_<название_тендера>_<timestamp>.json
tender_report_<название_тендера>_<timestamp>.md
```

## Получение API ключа

### Groq (БЕСПЛАТНО, рекомендуется):
1. Перейдите на https://console.groq.com/
2. Зарегистрируйтесь (бесплатно)
3. Перейдите в "API Keys"
4. Создайте новый ключ
5. Добавьте в `.env`: `GROQ_API_KEY=gsk_...`

### Другие провайдеры:
См. подробное руководство [LLM_PROVIDERS.md](LLM_PROVIDERS.md):
- OpenAI GPT (платный, дешевый)
- Google Gemini (бесплатный tier)
- Ollama (локально, без ключа)
- Anthropic Claude (оригинал, платный)

## Частые проблемы

### Ошибка: "API ключ не найден"
→ Проверьте, что файл `.env` создан и содержит правильный ключ для вашего провайдера
→ Убедитесь, что `provider` в `config/settings.yaml` соответствует вашему ключу

### Ошибка: "Файл не найден"
→ Укажите полный путь к файлу или убедитесь, что находитесь в правильной директории

### Ошибка: "Не удалось извлечь текст из PDF"
→ PDF может быть защищен или содержать только изображения. Попробуйте другой файл.

### Низкая оценка тендера
→ Проверьте настройки в `config/company_profile.yaml` — они влияют на скоринг

## Следующие шаги

1. Настройте веса критериев в `config/company_profile.yaml`
2. Отрегулируйте пороговые значения под свои нужды
3. Проанализируйте несколько тендеров для калибровки системы
